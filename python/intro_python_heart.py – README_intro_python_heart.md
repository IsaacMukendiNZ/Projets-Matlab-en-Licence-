# Introduction à Python et analyse du dataset Heart

## Description
Ce script combine des exercices d’initiation à Python (saisie, conditions, boucles, manipulation de chaînes) avec une analyse simple du fichier `TD1_heart.csv`.  
Il calcule des statistiques (moyenne, max, min, écart-type) sur l’âge et le cholestérol des patients, et affiche un histogramme comparant les distributions des patients sains et malades.

## Prérequis
- Python 3 avec les bibliothèques : `numpy`, `pandas`, `matplotlib`
- Fichier de données `TD1_heart.csv` dans le même dossier

## Utilisation
1. Placez le fichier `TD1_heart.csv` dans le dossier du script.
2. Exécutez le script :
   ```bash
   python intro_python_heart.py
