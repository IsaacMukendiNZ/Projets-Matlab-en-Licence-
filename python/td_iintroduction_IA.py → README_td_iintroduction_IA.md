# Travaux dirigés : Introduction à Python et au Machine Learning

## Description
Ce script regroupe de nombreux exercices progressifs :
- Bases de Python (variables, listes, dictionnaires, conditions, boucles, fonctions)
- Gestion d’erreurs (`try/except`)
- Utilisation de `numpy` (création de tableaux, manipulation, statistiques)
- Visualisation avec `matplotlib`
- Introduction à la régression logistique avec scikit-learn
- Premiers pas avec TensorFlow/Keras (réseau simple pour classification binaire)

C’est un support idéal pour apprendre ou enseigner les bases de la programmation scientifique en Python.

- Python 3 avec les bibliothèques : `numpy`, `matplotlib`, `scikit-learn`, `tensorflow`
- Aucun fichier de données externe n’est nécessaire (les données sont générées ou intégrées)
